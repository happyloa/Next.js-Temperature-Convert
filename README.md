# Next.js 14 溫度轉換器

一個用 Next.js 來製作的溫度轉換器。
線上 Demo：[點這裡](https://next-js-temperature-convert.vercel.app/)

## 如何安裝所需套件？

將檔案 clone 下來後，輸入：

    npm install

安裝需要的套件。

接著，輸入：

    npm run dev

在本地端啟動開發環境，預設的監聽端口是 3000，等到本地開發環境正式啟動後，可以在瀏覽器上輸入：

    http://localhost:3000

以在本地端預覽專案


### 使用的第三方套件：

- [Material UI](https://mui.com/material-ui/)
